# Personal Portfolio Website

## 📌 Description
This is a responsive personal portfolio website developed using HTML5, CSS3, and JavaScript.  
It presents my profile, hobbies, and projects in a clean and modern layout.  
The website also includes interactive features such as adding and deleting hobby benefits, form submission alerts, and a dynamic footer displaying the current date and time.

## 👨‍💻 Sections Included
- Header with name and tagline
- Navigation bar with internal section links
- About Me section
- Hobbies section (Gaming and Cricket)
- Projects showcase
- Contact form
- Footer with LinkedIn and GitHub links
- Dynamic date and time display

## 🚀 Features
- Responsive design using CSS media queries
- Modern layout using Flexbox and Grid
- Clean UI with Google Fonts
- Project cards with shadow and rounded design
- Styled contact form
- Add new hobby benefits dynamically
- Delete hobby items from the list
- Alert message when contact form is submitted
- Dynamic footer showing current date and time
- Mobile-friendly navigation

## 🛠 Technologies Used
- HTML5
- CSS3
- JavaScript
- Google Fonts
- Flexbox
- CSS Grid

## 📂 Projects Highlighted
1. Tic-Tac-Toe Game  
2. Frontend Recipe Website  
3. E-commerce Website (Ongoing)

## 📱 Responsiveness
The website is responsive and adjusts layout for screens smaller than 768px using media queries.
